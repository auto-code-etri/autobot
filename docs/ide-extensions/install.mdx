<Tabs>
<Tab title="VS Code">

<Steps>
<Step title="Install from Marketplace">
Click `Install` on the [Autobot extension page in the Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ETRI.autobot)
</Step>

<Step title="Install in VS Code">
  This will open the Autobot extension page in VS Code, where you will need to
  click `Install` again
</Step>

<Step title="Move to Right Sidebar">
The Autobot logo will appear on the left sidebar. For a better experience, move Autobot to the right sidebar

![move-to-right-sidebar](/images/move-to-right-sidebar-b2d315296198e41046fc174d8178f30a.gif)

</Step>

</Steps>

<Info>
  If you have any problems, see the [troubleshooting guide](/troubleshooting).
</Info>

</Tab>

<Tab title="JetBrains">

<Steps>
<Step title="Open Settings">
Open your JetBrains IDE and open **Settings** using `Ctrl` + `Alt` + `S`
</Step>

<Step title="Find Autobot Plugin">
  Select **Plugins** on the sidebar and search for "Autobot" in the marketplace
</Step>

<Step title="Install Plugin">
Click `Install`, which will cause the Autobot logo to show up on the right toolbar

![jetbrains-getting-started.png](/images/getting-started/images/jetbrains-getting-started-d62b7edee1cdd58508c5075faf285955.png)

</Step>
</Steps>

<Info>
  If you have any problems, see the [troubleshooting guide](/troubleshooting).
</Info>

</Tab>
